================================================================================
                            ASCIQUARIUM (C++/MFC)
================================================================================

Asciiquarium is a fun, animated aquarium rendered in ASCII art using Microsoft Foundation Classes (MFC). Enjoy watching fish, submarines, and other aquatic life swim across your Windows desktop!

--------------------------------------------------------------------------------
Features
--------------------------------------------------------------------------------
- Animated ASCII aquarium in a Windows dialog
- Multiple fish, submarines, and other marine life
- Colorful display (if supported)
- Simple, interactive interface

--------------------------------------------------------------------------------
Build Instructions
--------------------------------------------------------------------------------
1. Open `asciiquarium.vcproj` in Visual Studio.
2. Build the project (F7).
3. Run the resulting executable.

*Requirements:*
- Microsoft Visual Studio (tested with VC++ 2008 and newer)
- MFC libraries

--------------------------------------------------------------------------------
Usage
--------------------------------------------------------------------------------
- Launch the application.
- Watch the aquarium animation in the main dialog.
- Use the menu or buttons to interact (if available).

--------------------------------------------------------------------------------
File Structure
--------------------------------------------------------------------------------
- `asciiquarium.vcproj`      : Visual Studio project file
- `asciiquarium.h/.cpp`      : Main application class
- `asciiquariumDlg.h/.cpp`   : Main dialog class and logic
- `asciiquarium.rc`          : Windows resources (icons, dialogs, etc.)
- `res\asciiquarium.ico`     : Application icon
- `res\asciiquarium.rc2`     : Additional resources
- `StdAfx.h/.cpp`            : Precompiled headers
- `Resource.h`               : Resource IDs
- `asciiquarium.manifest`    : Application manifest
